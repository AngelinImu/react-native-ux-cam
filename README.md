# react-native-uxcam

## installation
`$yarn add file:/path-to-the-uxcam-react-wrapper`
`$react-native link react-native-uxcam`

## Usage
```javascript
import RNUxcam from 'react-native-uxcam';


RNUxcam.startWithKey('YOUR API KEY');
```
